PIMpoint Collaborator Portal User Manual

Collaborator Portal Pages

Summary Page


From the summary page, users can:

1. See sample tracking information

2. Print sample labels

3. Access package metadata

4. Submit package tracking information

Meta Data Page


Users can upload, edit and view sample metadata for their packages on the metadata page.

New Package Page


Users can create new packages with different amounts of samples on the create package page.

Error Page


Users can see and resolve all sample errors on the error page.

Contact Page


Users can contact TGen North about any issues or errors on the contact page.
